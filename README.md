# Sphere Invaders
Space Invaders for those who prefer their maps equidistant from the origin at all points.

![Screenshot of game](screenshot.jpg)

## How to Run
Install dependencies...
```
npm install
```

Build the project...
```
npm run build
```

Start a server...
```
npm run serve
```

Once the server is running, the game can be played at `localhost:8080`.
